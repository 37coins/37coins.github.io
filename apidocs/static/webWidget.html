Web Widget

Currently we have online registration for our SMS WALLET based on our WEBSITE. People generally go to the signup page, put their phone number and on form submission they are redirected to the website ACCOUNT page, where some server operations are done to get registered that phone number and the feedback of those operations are shown on that page whether the registration is done successfully or this is an unsuccessful operation and why. 37coins SMS WALLET signup web widget provides signup to 37coins free SMS WALLET from other websites. To enable sign-up to 37coins wallets on other websites a web widget has been created. This web widget can be integrated into other websites, social media etc where users can sign up there but they do not need to go to our website. This way much more people will get chance to use our SMS WALLET service.




Requirements

This is a jquery widget, need to include jquery if not present in the website.

  • jQuery 1.7.2+ (It is always recommended to use the latest version of jQuery) 
    <script src="http://ajax.googleapis.com/ajax/libs/jquery/1.11.1/jquery.min.js"></script>


  • The widget javascript FILE itself. 
    <script src="http://<the prod server>/widget/widget37coinsSignup.js"></script>

Integration

The widget can be initialized with a single call to a Javascript function

HTML

Create a div element with a unique id or class(The container which will hold the widget eg. "widget-container") 
<div id="widget-container"></div>

Javascript

Call the plugin: In your JavaScript code, call the widget37coinsSignup() method on your HTML div.

//Executes your code when the DOM is ready.  Acts the same as $(document).ready().
$(function() {
    $('#widget-container').widget37coinsSignup({
    //options go here
    });
});


Options

In this version of widget no options supported.
